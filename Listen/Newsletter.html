So nutzen Sie unseren Newsletter


  1. Registrieren Sie sich als neuer Benutzer. Klicken Sie dazu auf der Startseite auf den Link Regestrieren.
  2. Geben Sie Ihren Namen und Ihre e-mail-Adresse ein.
  3. Aktivieren Sie das Kontrollkästchen für Newsletter.
  4. Wenn Sie nur den Newsletter bestellen wollen, klicken Sie auf Abschicken.

Wenn Sie sich zusätzlich für unser Userforum registrieren lassen wollen:

  1. Aktivieren Sie zusätzlich das Kontrollkästchen Userforum
  2. Vergeben Sie ein persönliches Kennwort. Um Tippfehler zu vermeiden, müssen Sie das Kennwort zweimal eingeben.
  3. Klicken Sie auf Abschicken
  4. Sie erhalten kurze Zeit später Ihren persönlichen Zugangscode per e-mail.
